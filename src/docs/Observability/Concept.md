# Concept for Observability

## Зони віповідальності

- Faro/Sentry: Помилки.

- Prometheus: Метрики/Тренди.

- Loki: Логи/Контекст.

- Tempo/Alloy(Tempo)/Jaeger: Трейси/Швидкодія.

- K6: Тестування

## Grafana Cloud

На відміну від Sentry (який є комерційним SaaS-продуктом), **Prometheus, Loki, Jaeger та OpenTelemetry — це Open Source проекти**.

Компанія **Grafana Labs** (розробники Grafana і Loki) має хмарну платформу, де все це вже налаштовано. Це аналог Sentry, але для всього стеку.

**Free Forever** план:

| Інструмент               | Безкоштовний ліміт (на місяць) | Коментар архітектора                                                        |
| ------------------------ | ------------------------------ | --------------------------------------------------------------------------- |
| **Grafana** (UI)         | **3 користувачі**              | Достатньо для тебе + бекендера + PM.                                        |
| **Metrics** (Prometheus) | **10,000 серій**               | Це багато. Для середнього пет-проекту або MVP стартапу вистачить з головою. |
| **Logs** (Loki)          | **50 GB**                      | Це _дуже_ багато. Текстові логи легкі. 50 Гб вистачить на мільйони запитів. |
| **Traces** (Tempo\*)     | **50 GB**                      | Теж дуже щедро.                                                             |
| **K6** (Testing)         | 500 віртуальних юзер-годин     | Для навантажувального тестування.                                           |
